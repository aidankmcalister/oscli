# @oscli-dev/react

React components for `oscli` demos and previews.

`OscliDemo` replays an `oscli` CLI in React.

> **Warning:** This feature is experimental. It can break, differ from terminal
> behavior, or fail in some setups.

It uses the same CLI instance you already ship. If you register `cli.main()`,
the demo follows that handler in order.

## Install

Install the React package with the core package.

```bash
npm install @oscli-dev/react @oscli-dev/oscli
```

You also need `react` 18 or newer.

## Render it

Pass a CLI instance to `OscliDemo`. Give the parent element a fixed height.

```tsx
import { OscliDemo } from "@oscli-dev/react";
import { cli } from "./cli";

export function Demo() {
  return (
    <div style={{ height: 320 }}>
      <OscliDemo cli={cli} />
    </div>
  );
}
```

## Run CLI code

Use `cli.main()` to register the handler separately so `OscliDemo` can replay
it via `cli.animate()`.

```tsx
import { createCLI } from "@oscli-dev/oscli";

export const cli = createCLI((b) => ({
  title: "create-app",
  prompts: {
    project: b.text().label("Project").default("my-app"),
  },
}));

// Register handler for OscliDemo replay
cli.main(async () => {
  await cli.prompt.project();
  await cli.spin("Scaffolding project", async () => {});
  cli.success(`Created ${cli.storage.project}`);
});
```

## Props

These are the main props.

- `cli`: Your CLI instance.
- `answers`: Pins answers. Unpinned answers are generated by the demo.
- `replay`: Runs once, forever, or a fixed number of times.
- `speed`: Uses the built-in `slow`, `normal`, or `fast` preset.
- `theme`: Uses `auto`, `light`, `dark`, or custom tokens.
- `fade`: Uses `false` or a duration in milliseconds.
- `script`: Adds manual steps when you need overrides.
- `className`, `style`, `onRunComplete`: Rendering and lifecycle hooks.

## Pin answers

Pass `answers` when you want fixed values. Set `replay={false}` to run once.

```tsx
<OscliDemo
  cli={cli}
  answers={{
    project: "my-app",
    framework: "next",
    approved: true,
  }}
  replay={false}
/>
```
